# 📒 おてつだい手帳

親子で使えるお手伝い報酬管理アプリ。QRコードでごほうびをやり取りします。

## 🎯 概要

- **親アプリ**: お手伝い項目・金額を設定してQRコードを生成
- **子アプリ**: QRコードを読み取ってごほうびを受け取り、残高を管理
- **サーバー不要**: すべてのデータは端末のローカルストレージに保存
- **PWA対応**: Androidにアプリとしてインストール可能

## 🚀 使い方

### セットアップ
1. 子どもの端末で「子アプリ」を選択 → 名前とアバターを設定
2. 子どもの「マイQR」タブのQRコードを表示
3. 親の端末で「親アプリ」を選択 → 「こどもを追加」で子どものQRをスキャン

### お手伝いの報酬
1. 親: 子どもを選択 → お手伝い項目を選択 → 金額を調整 → QRを生成
2. 子: 「ごほうびQRをよみとる」でQRをスキャン → 残高に加算！

## 🔒 不正防止

- 報酬QRにはSHA-256ハッシュが付与されており、改ざんを検知
- QRコードは特定の子どものIDに紐づいているため、別の子どもでは受け取り不可
- 連番（seq）により同一QRの再利用を防止

## 🛠 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- [qrcode.js](https://github.com/davidshimjs/qrcodejs) - QRコード生成
- [html5-qrcode](https://github.com/mebjas/html5-qrcode) - QRコードスキャン
- PWA (Service Worker + manifest.json)

## 📦 デプロイ

3. `https://<username>.github.io/<repo-name>/` でアクセス可能

## 📱 インストール（Android）

1. Chrome でアプリのURLを開く
2. メニュー（︙）→「ホーム画面に追加」or「アプリをインストール」
3. ホーム画面にアイコンが追加される
